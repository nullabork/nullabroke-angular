# v0.1.0

## New Features
- Query builder with dynamic parameter syntax ({Label:Type:Default})
- Four parameter types: StringInput, NumberInput, FormTypes dropdown, Tags multi-select
- Sidebar with saved queries, auto-save, and query management
- 19 default blueprint queries for common SEC filing searches
- Document viewer with iframe-based rendering
- Pin/unpin queries to keep favourites at the top
- Middle-click queries to open in a new tab
- Help page with query cheat sheet and full reference
- Welcome modal with tips for new users

## Improvements
- Queries sorted by most recently used
- Blueprint queries auto-detach when edited
- Restore Defaults brings back deleted blueprints without touching modified ones
- Filing descriptions shown in search results

## Infrastructure
- Docker deployment with nginx (Coolify-ready)
- GitHub Actions CI pipeline with build and test
- Auth0 authentication with Google and GitHub login support
